Esto es un sitio web para
informar y promover el uso de
este y otros estilos de bicicleta
entendiendo que es un deporte que
reune características aptas para la
vida en ciudad y compatible con las
actividades de hoy en dia. Sin duda
queda mucho por delante que se respete
el lugar que ocupa la bicicleta en el
transporte y en la cultura, aqui acercamos
un poco de este mundo.



Estilo preferido:











Hoy hablaremos sobre un sistema de rodamiento
bastante antiguo, que se utiliza oficialmente para las
carreras de las olimpíadas, dentro de un velódromo
y que hace mas de 50 años, aficionados se atrevieron
a llevarlo a la calle. Se trata de bicicletas, con piñon fijo
y sin frenos.
